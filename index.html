FppShoplist

own project C++ QT
FppShoplist is an addition to FPP Firma++ (warehouse management programme) and is intended to extend the basic functions of FPP to more computer stations. FppShoplist allows to operate on items managed in FPP by importing a FPP export file to another terminal. After the import user has permanent access to all items and their stats. Items can be simply searched or inserted into main list, as if a receipt or invoice were made. The list can be transferred via the Internet to FppShoplist server and inserted into real receipt or invoice in FPP Firma++.
